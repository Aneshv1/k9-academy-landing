# K9 Academy — Monthly Email Campaigns

## How to Use These

Each month has 16 emails (3-4 per week). Each file contains:
- Subject line
- Preview text
- Format (graphic or plain-text)
- Pillar (educational, social proof, brand, service spotlight, or offer)
- Full email copy

## Schedule

Send 3-4 emails per week. Suggested days: Monday, Wednesday, Friday (+ Thursday when 4 per week).

## Rules

- Never discount. Never hint at deals. Full price, always.
- Plain-text emails: no HTML, no images, no branding. Just text. Sign off as "Anesh."
- Graphic emails: use K9 Academy branding, include images/infographics where noted.
- Sender: K9 Academy <contact@k9academy.ca>
- Reply-to: contact@k9academy.ca
- No em-dashes in any copy.
- Keep body copy under 200 words where possible.

## Monthly Pillar Balance

| Pillar | Count | Format |
|--------|-------|--------|
| Educational | 4 | 3 graphic, 1 plain-text |
| Social Proof | 3 | 2 graphic, 1 plain-text |
| Brand/Community | 3 | 1 graphic, 2 plain-text |
| Service Spotlight | 4 | 3 graphic, 1 plain-text |
| Offer/Sale | 2 | 2 graphic |
| **Total** | **16** | **11 graphic, 5 plain-text** |
